# Pro Fashion Studio AI App

This project was configured to run with Vite and React.

## Prerequisites
- Node.js (v18 or higher)
- npm (usually comes with Node.js)

> **Note**: An installation of Node.js via Homebrew (`brew install node`) was initiated. Please verify it completes successfully before running the commands below.

## Setup
Install the necessary dependencies:

```bash
npm install
```

## Running the App
Start the development server:

```bash
npm run dev
```

The app will naturally be available at `http://localhost:5173`.
